# Lune — Sales Demo

Interactive mobile demo landing for **Lune** (transaction intelligence for banks & fintechs).

Toggle **Without Lune** / **With Lune** to see raw bank transactions transform into
enriched merchants, logos, categories and insights — with live donut, category &
brand trends, spending-over-time and a real date-range filter (30d / 3mo / 6mo / 12mo).

Single self-contained `index.html` (React + Babel via CDN). Open it in a browser,
or serve statically / enable GitHub Pages.
